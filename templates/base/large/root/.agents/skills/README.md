# Skills

These skills are optional large-mode enhancement templates.
They do not execute anything by themselves.
They are not required for small mode.
They are not automatically installed globally.
They are not third-party skills.
They are local project-level templates generated by this harness.
If an environment does not support skills, the main Codex agent should read AGENTS.md and docs/ai/* directly.
Human review gates remain authoritative.

Do not install unreviewed third-party skills directly into this directory.
Do not put secrets or environment-specific credentials in SKILL.md.
Do not use skills to bypass review gates.

Current skill groups:

- `methodology/`
- `system/`
