# Prompts

This directory stores prompt source used by the harness.

Planned groups:

- `bootstrap-local-agent.md`
- `planning/`
- `review/`
- `handoff/`
- `subagents/`
- `core/`

Prompt source belongs here.
Prompt-generated runtime output belongs in target projects.

`bootstrap-local-agent.md` is a one-time setup prompt.
It tells the active local agent how to inspect this harness and install only the instruction and skill sources its own environment supports.
