# Examples

This directory contains generated target-project examples and companion sample
repositories.

Use it to show:

- `small/`: output from `ai-init small`
- `large/`: output from `ai-init small` followed by `ai-upgrade large`
- `cpp-linux-backend-mini/`: a tiny pre-injection business repository
- `injected-output-sample/`: the same style of repository after harness
  injection and large-mode task setup
- overlay results for `cpp-linux-backend-system`

Do not treat example output as the template source of truth.
The source of truth remains `templates/` plus `profiles/`.
