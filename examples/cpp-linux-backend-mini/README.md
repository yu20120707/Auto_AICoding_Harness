# cpp-linux-backend-mini

This is a tiny C++ / Linux / backend-oriented sample repository for exercising `Auto_AICoding_Harness`.

Use it to dry-run the normal adoption flow:

1. run `ai-init small`
2. upgrade to `medium` or `large` when the task expands
3. inspect generated `AGENTS.md`, `docs/ai/*`, `.ai/state.json`, and task artifacts

This sample is intentionally small:

- no third-party dependencies
- one tiny library target
- one tiny executable
- one lightweight test file

It is a business-project example, not a harness template source.
